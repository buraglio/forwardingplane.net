BCM56870

Release: Sept 4 2014

Edge-corE appears to be a wholey owned subsidiary of Accton. Accton is an ODM -- Original Device Manufacturer. They make stuff for other companies. This switch is claimed to be the first ONIE switch certified by the Open Compute Project. Both Cumulus and Pica8 have certified the AS5712-54X to run their software. This switch was used as an example of relative cost of whitebox v traditional switch costs by JR Rivers. The original blog is https://cumulusnetworks.com/blog/the-facts-behind-the-myth/ for those that want the links to be active. Switch processor is Intel.

2018 Update

The 2018 version of the data sheet has added to the software compatibility list:

  • ONL Open Network Linux
  • Big Switch Big Monitoring Fabric and Big Cloud Fabric
  • Cumulus Linux r2.5.1 and later
  • PicOS r2.6 and later
  • SnapRoute FlexSwitch
  • Pluribus Netvisor OS version 2.5 and later
