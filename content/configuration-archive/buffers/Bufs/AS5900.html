EdgeCore AS5900-54 or AS5912-54

The model number seems to have changed slightly between first light to OCP and actual product. This design was submitted to OCP in March 2016. There is an OCP specification.

The Qumram has 8 packet ram chips, each with 32M x 16 bits-wide. The memory part number is K4A4G165WD by Samsung.

Update

There is now a data sheet as of September 2017. A more recent data sheet is more specific about the size of the external packet memory [6GB]. Info on the opencompute.org site seems to say that there is now a varient with TCAM. Presumably that means external TCAM since it seems there must be TCAM.
