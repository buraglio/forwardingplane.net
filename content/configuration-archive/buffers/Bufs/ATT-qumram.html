AT&T; open spec for XGS-PON

Does this really belong here? The device described here provides interworking between Ethernet and PON. But it uses a Broadcom Qumram switch chip. There is no off-chip buffer memory attached to the switch chip -- but there is memory associated with each of the line ASICs that drive the XFP interfaces. There is a lot of good information on operation of PON in the AT&T; open source submission.
