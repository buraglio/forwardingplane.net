OpenVswitch 2.3.0 RPMs

October 10, 2014

I was wanting to do a few quick mock-ups with OpenvSwitch and OpenDayLight and wanted to use CentOS since I have templates for it that I replicate.  Just like with the debian stuff I had been doing, I wasn’t able to find any in some quick searches.   I stumbled upon This site, which had a great how to for building them, so I just used that.  Seeing as that the debian packages actually got downloaded a lot, I figured I’d post these RPMs as well.  They’re available here and should work on CentOS 6.5 and probably 7. Just do a quick local install and start the service and you should be up and running.   yum localinstall /home/ovswitch/rpmbuild/RPMS/x86\_64/openvswitch-2.3.0-1.x86\_64.rpm  sudo service openvswitch start
