Edge-Core AS5916-54XL

This is the version of the AS5912 that has external (extra) TCAM for more routes. There is a data sheet.

  • Switching Capacity: 800 Gbps
  • Processing Rate: 700 Mpps
  • Jumbo Frame: 9 Kbytes
  • 16MB on-chip Buffering
  • Deep DRAM Buffering: 8 GB

Broadcom Qumran-MX BCM88370

Curiously, Edge-core claims that this switch can run IOS-XR. It is a close cousin to the NCS5501, but not exactly the same.

The AS5916-54XL is a version of the XKS without TCAM.

A more recent datasheet from May 2020 adds the new AGR110 model designation.
