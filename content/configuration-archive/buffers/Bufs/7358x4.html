Arista 7358X4 8-slot Trident-4 switch

Introduced November 2021
The switch has a single Broadcom Trident-4 ASIC with 32 lanes permanently routed to each of the slots. Each lane can be independently set to 28 or 56 Gb/s. The lanes can also be run slower (10 Gb/s) if necessary.

The port table from the datasheet is reproduced here:
Interface Modules DCS-7368-16S DCS-7358-16C DCS-7368-4D DCS-7368-4P
Ports16 x SFP2816 x QSFP284 x QSFP-DD4 x OSFP
Max 400GbE Ports — — 4 4
Max 200GbE Ports — 8 8 8
Max 100GbE Ports — 16 16 16
Max 25GbE Ports 16 16 — —
Max 10GbE Ports 16 16 — —

The four port 400G cards will need 8 56G lanes per port. So 32 lanes for the card.

The 16-port QSFP28 card needs 4 28G lanes per port. So 64 lanes for the card. That implies that there must be gearbox ASICs to convert 56G from the ASIC to match the QSFP28 requirements.
