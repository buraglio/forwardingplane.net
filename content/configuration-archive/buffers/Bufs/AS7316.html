Product introduction

As noted, October 1, 2018

This is a cell site gateway which means it has hardware for IEEE1588 timing and deep buffers. 48V DC power supplies are shown. AC power supplies are available. There is a data sheet. An OCP Switch Specification was part of the design package. The slides that accompanied the submission note that SONiC, Trellis and Stratum are supported on this hardware.

A version of the datasheet dated Jan 23, 2022 lists the following:

  • Compatible with ADVA Ensemble Activator
  • Compatible with CapGemini SDvAS
  • Compatible with Exaware ExaNOS
  • Compatible with Infinera CNOS
  • Compatible with IP Infusion OcNOS
  • Compatible with RtBrick Full Stack OS
  • Compatible with Volta Networks VEVRE
  • Compatible with Open Network Linux (ONL)
